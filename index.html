mod_k2_related_items

List related items for K2 @Javi_Mata

This module try get the actual categories to display a list of related items for category

First, get the Menu value, if this is a K2 Categories get the categories params Second, if use a module like K2 Tools get the category for the link Thirt, if are in a Item view get the id of the item and get the category
